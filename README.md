#Ecommerce app

Ruby on rails Ecommerce


###Payments

- Stripe
- Paypal

You need two env variables for stripe:

- ENV["STRIPE_PUBLIC"]
- ENV["STRIPE_SECRET"]

[SignUp in stripe](https://stripe.com/)

And another two for paypal:

- ENV["PAYPAL_PUBLIC"]
- ENV["PAYPAL_SECRET"]


[SignUp in paypal](https://developer.paypal.com/)
